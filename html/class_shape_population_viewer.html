Detailed Description

The ShapePopulationViewer class.

ShapePopulationViewer Gui class specification. This class contains all model data and controller callbacks, if we are going to consider the code within the MVC paradigm. See ui_ShapePopulationViewer.h for information on the construction of the gui itself. That is an autogenerated file from the ShapePopulationViewer.ui file, which could also be used as reference.

Michael Guarino

ShapePopulationViewer::viewChange.

Generally, <x,y,z> is a distance-normalized vector specifying the position you want to place the viewing camera (distance being the current distance from the camera to the mesh).
